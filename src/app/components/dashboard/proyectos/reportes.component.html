Portfolio Personal

Este Proyecto Proyecto desarrollado en el marco del Programa Argentina Programa. Proyecto SPE Desarrollo en Angular, Base de Datos en Mysql , Backend Java Usando SpringBoot y ApiRest. El proyecto tiene Seguridad JWT con la implementación de usuarios, En la visualización inicial está disponible el material del portfolio, y es editable con permiso de usuario realizando el login como Administrador. Se utiliza Firebase Storage para almacenar las imágenes, Se implementó la Base de Datos MySQL en Clever Cloud, Heroku para alojar el Backend. y Firebase Hosting para el Frontend.

Muestra de Data Analisis de Exportaciones Vitivinícolas Argentinas Tableu Public

Ver Proyecto en Tableau El proceso de ETL comienza con la extraccíon y procesamiento de distintas fuentes de datos e integrando a una base principal en la cual se ha trabajado mas de 10 años en la investigación y clasificación de marcas y variedades exportadas desde Argentina al Mundo. Me sumé al proyecto en el año 2009 y desde ese momento comencé con el análisis y procesamiento de datos de Exportaciones Vitivinícolas de Chile investigando e incluyendo mas de 6000 marcas se hizo cleansing de datos de años anteriores desde el 2017 y actualmente continúo agregando procesos de clasificación limpieza e integración de datos de estas fuentes de información. Los usuarios finales son Export manager e interesados del Mercado del Vino de todo el Mundo. (la Muestra es solo una representación de datos estadídsticos en cumplimiento de los derechos comerciales de laCaucasia Wine Thinking)
